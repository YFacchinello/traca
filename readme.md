# 📚 Traça

## Sobre o projeto

O Traça é um aplicativo desenvolvido para incentivar o hábito da leitura por meio da gamificação. A proposta é transformar a leitura em uma experiência mais interativa, permitindo que usuários acompanhem seu progresso, estabeleçam metas e, futuramente, participem de desafios e grupos com amigos.

O projeto nasceu durante meus estudos em Análise e Desenvolvimento de Sistemas com o objetivo de aplicar conceitos de Engenharia de Software, modelagem de domínio, desenvolvimento back-end e boas práticas de arquitetura em um software desenvolvido do zero.

## Objetivos do projeto

* Incentivar o hábito da leitura.
* Tornar a leitura uma experiência mais social.
* Desenvolver um sistema escalável utilizando boas práticas.
* Construir um projeto completo para portfólio.

## Tecnologias previstas

* PostgreSQL
* JPA/Hibernate
* JWT
* Docker
* Git e GitHub
